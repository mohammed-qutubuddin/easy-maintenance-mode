== Description ==
A super simple WordPress plugin puts your website into maintenance or under construction mode - along with displaying a countdown timer. It's possible to set your maintenance message, logo and end date/time from the plugin's settings page. No bloat, no confusion - Surf One Click Me On/Off!

== Installation ==
1. Download the zip file of the plugin.
2. Go to your wordpress site click on > Add New > Click on > Upload Plugin.
3. Upload the zip file and activate the plugin
4. Settings > Configure your maintenance mode (message, logo, date/time).

== Frequently Asked Questions ==
= Is it free? =
Yes it's totally free No hidden charges.

= Which role will maintenance mode look at? =
Only new users (regular visitors) will see the maintenance page. Logged-in users will see the normal site.

= Can I add a custom logo or message? =
Yes, you can upload/select any message and your image URL from the settings page.

= What happens after the countdown? =
When the timer expires, maintenance mode does not auto-off, but the message “Maintenance Complete!” is displayed.
